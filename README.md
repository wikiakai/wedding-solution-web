## Quick Start

- Clone the repo: `git clone https://github.com/wikiakai/wedding-solution-web.git`
- Install dependencies: `npm install` or `yarn`
- Start the server: `npm run start` or `yarn start`
- Views are on: `localhost:3000`

## Tech Stack
- React : 18.2.0
- Material UI : 5.11.0
